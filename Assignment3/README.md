# Assignment 3 - Small Object Detection Using YOLO

This assignment implements small object detection for traffic lights using YOLOv8 on the Bosch Small Traffic Lights Dataset (BSTLD).

## Dataset
- Bosch Small Traffic Lights Dataset (BSTLD)
- https://hci.iwr.uni-heidelberg.de/node/6132

## Folder contents
- `scripts/` : dataset preparation, baseline evaluation, trained evaluation, threshold/NMS sweep, and output generation scripts
- `submission/code/` : final code included for submission
- `submission/deliverables/` : annotated samples, baseline results, dataset info, final predictions, trained model, trained results, and training artifacts
- `report.pdf` : final report submitted to Canvas
